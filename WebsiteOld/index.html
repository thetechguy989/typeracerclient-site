TypeRacer Client

Non-injectable JS code.

PyQt5 technology disallows injecting JS code into the console, which makes cheating much harder to do.

Lightweight.

The TypeRacer Client has been heavily optimised for running TypeRacer and it still runs perfectly single-threadedly. This means that it can even run on a single-core PC.

You can invert it's colors.

Can be used as Dark Mode for TypeRacer. A requested feature that comes with the TypeRacer Client.
